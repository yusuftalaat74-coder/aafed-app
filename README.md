# AAFED App — تطبيق أفريقيا للتعليم والتنمية

منصّة تربط المتبرّع بالمشروع وبالمستفيد، بشفافية وتوثيق ولمسة إنسانية.
المتبرّع يتابع مشروعه من اختيار المنطقة حتى التشغيل، عبر تايم لاين وتقارير متابعة موثّقة.

## التقنية (Stack)

- **التطبيق (موبايل):** Flutter — كود واحد لأندرويد و iOS.
- **الباكند:** Node.js + Express.
- **قاعدة البيانات:** PostgreSQL.

## هيكل المستودع (Monorepo)

```
aafed-app/
├── mobile/      # تطبيق Flutter (المتبرّع)
├── backend/     # Node.js + Express + PostgreSQL API
└── README.md
```

## التشغيل المحلي

### الباكند
```bash
cd backend
cp .env.example .env      # عدّل بيانات قاعدة البيانات
npm install
npm run db:init           # ينشئ الجداول من db/schema.sql
npm run dev               # يشغّل API على http://localhost:4000
```

### التطبيق (Flutter)
```bash
cd mobile
flutter pub get
flutter run               # على محاكي أو جهاز
```

## الهوية البصرية (AAFED)

- كحلي أساسي: `#0E3449` / `#16475C`
- أخضر: `#2E7D43` / `#7CB342` / لايم `#8DC044`
- الاتجاه: RTL (عربي) مع دعم متعدد اللغات لاحقاً.

## خريطة المراحل

- **المرحلة 1 (الحالية):** الهيكل + شاشة الرئيسية + API المشاريع.
- المرحلة 2: تايم لاين المشروع + التوثيق + المتبرّع (CRM).
- المرحلة 3: الكفالات + الدفع + لوحة الإدارة.
