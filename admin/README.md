# AAFED Admin — لوحة الإدارة (ويب)

لوحة تحكّم تستهلك الـ API وتعرض كل شيء في مكان واحد.

## التشغيل
1. شغّل الباكند: `cd backend && npm run dev`
2. افتح `admin/index.html` في المتصفح (أو قدّمه عبر أي خادم ثابت).
3. لو الـ API على عنوان آخر، عدّله من خانة "API" أعلى اللوحة.

## الأقسام
- **نظرة عامة**: مؤشرات + حالة كل مشروع (تقدّم/تأخّر) + تنبيهات التأخّر.
- **التنبيهات**: تقرير/تبرّع/بدء مرحلة/خطر تأخّر.
- **القسم الإعلامي**: محتاج تصوير · صُوِّر ولم يُنشر.
- **المتعاونون (CRM)**: الأفراد والجمعيات ومشاريعهم.
- **أخبار موزمبيق**: النبض العام.
- **القصص والأخبار**: القصص الإنسانية واقتراحات الأخبار.
- **تقييم المتبرّعين**: تقييم الجمعية وتقاريرها + اقتراحات التعديل.
